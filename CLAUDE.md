# 贝叶斯公式交互演示

一个单文件 HTML 演示页面，用可拖拽的二维参数空间帮助理解贝叶斯公式。适合教学、自学或嵌入到更大课程中作为交互组件。

## 已实现功能

- **二维可拖拽参数空间**
  - 左右拖动：调整先验概率 `P(H)`
  - 上下拖动：调整数据概率 `P(D)`
  - 由于 `P(D)` 受 `P(H)` 与 `P(D|¬H)` 约束，拖动时会被自动限制在可行域内
- **实时公式展示**
  - 使用真正的分数线，等号与分数居中对齐
  - 第一行：`P(H|D) = P(D|H)·P(H) / P(D)`
  - 第二行单独展示数据概率的全展开：`P(D) = P(H)·P(D|H) + P(D|¬H)·P(¬H)`
- **可视化贝叶斯元素**
  - 蓝色区域：`P(H)`
  - 绿色区域：`P(D|H)·P(H)`（分子）
  - 红色区域：`P(D|¬H)·P(¬H)`（分母中的另一项）
  - 浅灰色三角形：`P(D)` 在给定 `P(D|¬H)` 下的可行域
- **动态指标与后验条**
  - 实时显示 `P(H)`、`P(D)`、`P(D|H)`、`P(D|¬H)`、`P(H|D)`
  - 用颜色渐变条直观展示后验概率大小
- **辅助控制**
  - 滑块调节 `P(D|¬H)`
  - 明暗主题切换
  - 响应式布局，适配窄屏

## 如何友好地演示贝叶斯公式

### 核心教学叙事

1. **先建立直觉**：先验 `P(H)` 是我们在看到数据前对假设的相信程度。
2. **引入数据**：`P(D|H)` 是假设成立时观察到数据的可能性；`P(D|¬H)` 是假设不成立时观察到数据的可能性。
3. **解释数据概率**：`P(D)` 是无论假设是否成立，观察到数据的总概率。它既可以用一个符号表示，也可以展开成 `P(H)·P(D|H) + P(¬H)·P(D|¬H)`。
4. **得出后验**：后验 `P(H|D)` 就是“支持 H 的证据”在“所有证据”中所占的比例，即绿色区域面积占绿+红总面积的比例。

### 演示时可以强调的互动点

- 把 `P(D|¬H)` 调得很低，再让 `P(D)` 保持较高：后验会显著升高，因为数据很难用反假设解释。
- 把 `P(H)` 拖到很小：即使 `P(D|H)` 很高，后验也可能不高，因为先验太弱。
- 把 `P(D)` 拖到可行域上/下边界：此时 `P(D|H)` 达到 1 或 0，是极端情况。

### 可视化映射

| 视觉元素 | 数学含义 |
|---------|---------|
| 蓝色区域面积 | `P(H)` |
| 绿色区域面积 | `P(H)·P(D|H)`（分子） |
| 红色区域面积 | `P(¬H)·P(D|¬H)` |
| 绿+红区域面积 | `P(D)` |
| 绿色面积 / (绿+红) | `P(H|D)` |

## 技术栈

- **纯前端**：单个 HTML 文件，无构建工具、无依赖
- **SVG**：用于绘制参数空间、彩色区域、坐标轴和可拖拽手柄
- **原生 JavaScript (ES6+)**：所有交互逻辑
- **CSS 变量**：统一控制浅色/深色主题和配色
- **无框架**：便于直接嵌入任何静态站点或课程页面

## 界面展示风格

### 设计原则

- **清晰优先**：公式、指标、图表都用颜色编码一一对应
- **颜色语义固定**：
  - 蓝色 = 先验 `P(H)`
  - 绿色 = 似然 `P(D|H)`
  - 红色 = 反假设似然 `P(D|¬H)`
  - 青色 = 数据概率 `P(D)`
  - 紫色 = 后验 `P(H|D)`
- **卡片式布局**：左侧参数空间，右侧公式与指标
- **轻量阴影与圆角**：现代、友好的视觉感受
- **明暗主题**：默认跟随系统偏好，可手动切换
- **无障碍考虑**：
  - 文字与背景对比度充足
  - 标签使用背景药丸避免与彩色区域混淆
  - 图表元素使用 `pointer-events="none"` 避免干扰拖拽

### 响应式

- 桌面：左右双栏
- 平板/手机（≤820px）：单栏堆叠
- SVG 使用 `viewBox`，可随容器自动缩放

## 如何运行与继续开发

```bash
open bayes-demo.html
```

python3 -m http.server 8000

## 数学模型说明

页面内部状态：

- `prior` = `P(H)`，由水平拖动控制，范围 `[0, 1]`
- `dataProb` = `P(D)`，由垂直拖动控制
- `falseLikelihood` = `P(D|¬H)`，由滑块控制，范围 `[0, 1]`
- 推导 `likelihood` = `P(D|H)` = clamp((`P(D)` - `(1-P(H))·P(D|¬H)`) / `P(H)`, 0, 1)

`P(D)` 的可行域：

```
(1 - P(H)) · P(D|¬H)  ≤  P(D)  ≤  P(H) + (1 - P(H)) · P(D|¬H)
```

后验：

```
P(H|D) = P(D|H) · P(H) / P(D)
       = P(D|H) · P(H) / [P(H)·P(D|H) + P(¬H)·P(D|¬H)]
```

## 注意事项

- 当 `P(H) = 0` 时，`P(D|H)` 无定义，页面将其设为 `P(D|¬H)` 作为极限处理；此时绿色区域面积为 0。
- 所有概率值在 UI 中保留两位小数，后验条使用百分比整数显示。
- 主题切换通过 `data-theme` 属性和 CSS 变量实现，不使用系统自动翻转（用户可手动选择）。
